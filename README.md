# Своя Игра

Браузерная версия викторины «Своя Игра» для компании друзей. Работает полностью статически — без сервера, прямо на GitHub Pages.

**Играть:** https://aslyamov.github.io/jeopardy/

## Возможности

- Создание собственных наборов вопросов через встроенный редактор
- Поддержка картинок по URL
- Импорт и экспорт наборов в формате JSON
- Настраиваемое количество игроков
- Таймер на вопрос (опционально)
- Сохранение и возобновление игры через localStorage
- Встроенный набор вопросов для быстрого старта

## Как играть

1. Откройте сайт
2. Нажмите **Новая игра**
3. Выберите набор вопросов, введите имена игроков, настройте таймер
4. На игровом поле выбирайте вопросы по категориям и стоимости
5. После раскрытия ответа отмечайте игроков, ответивших правильно
6. В конце игры — экран с итогами и рейтингом

## Стек

- TypeScript + esbuild
- Tailwind CSS (CDN)
- localStorage для хранения данных
- GitHub Pages для хостинга
